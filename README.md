## Run Locally

Clone the project

```bash
  git clone https://github.com/livingdolls/twitter-material.git
```

Go to the project directory

```bash
  cd twitter-material
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
